Tabby the Task Bot

Tabby is a command line task bot written in Java. It is used to maintain a task list consisting of various categories of tasks, such as to-dos, deadline and events.
The GUI of Tabby is build in JavaFX and allows users who are unfamiliar with command line to interact with Tabby.
The link below leads to the project website, which includes a user guide documenting all available Tabby commands. The GitHub repository can also be found here.
